## 待处理工作

显示需要用户处理的日清工作，包括项目活动、流程工作、日常工作、临时工作。

### 日常工作日清

日常工作，每天只能添加一次日常工作。

日清助手的内容，来自岗位关联的岗位职责。


### 临时工作日清

临时工作，只能在完成时提交一次，不可分次完成。
